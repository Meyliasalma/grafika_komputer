Judul proyek : Space Defender Game 
Space Defender Game adalah game 2D sederhana berbasis Python dan Tkinter. Game ini dibuat sebagai proyek Grafika Komputer untuk menerapkan algoritma grafika dasar, transformasi geometri 2D, serta animasi real-time dalam bentuk game interaktif.
Pemain berperan sebagai pilot pesawat luar angkasa yang harus menghancurkan musuh dan mengumpulkan skor hingga 500 poin untuk memenangkan permainan.
Konsep grafika dan algoritma yang digunakan :
- Polygon sebagai objek dasar pembentuk pesawat dan musuh
- Algoritma DDA (Digital Differential Analyzer) untuk menggambar sisi polygon
- Algoritma Midpoint Circle untuk menggambar objek berbentuk lingkaran
- Transformasi Geometri 2D (translasi, rotasi, skala, dan refleksi)
- Animasi dan game loop real-time
Cara Menjalankan dan Memainkan Game:
- Pastikan Python versi 3.x telah terinstall pada komputer.
- Buka folder proyek Space Defender Game.
- Jalankan program melalui terminal atau command prompt dengan perintah:
   python main.py
- Setelah program dijalankan, jendela game akan terbuka dan menampilkan menu awal.
- Tekan tombol START untuk memulai permainan.
