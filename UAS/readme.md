1.Judul Proyek: Visualisasi Ruangan 3D Sederhana Menggunakan Transformasi dan Proyeksi Perspektif
2.Konsep Grafika yang Digunakan: Program ini menerapkan konsep dasar grafika komputer 3D, meliputi:
Transformasi 3D: translasi untuk penempatan objek, rotasi pada kamera (first person view), skala berdasarkan ukuran vertex, dan refleksi pada lantai.
Viewing: menggunakan sudut pandang kamera orang pertama (First Person View).
Proyeksi 3D: menggunakan proyeksi perspektif sehingga objek yang lebih jauh terlihat lebih kecil.
Ilusi Kedalaman: dihasilkan dari perbedaan ukuran objek berdasarkan jarak dan pengurutan objek berdasarkan nilai kedalaman (Z).
3.Cara Menjalankan Program: Pastikan Python telah terinstal di komputer,Buka folder proyek menggunakan editor atau terminal (misalnya Thonny atau VS Code).
Jalankan file utama program dengan perintah:python main.py,Setelah program berjalan, gunakan keyboard dan mouse untuk menggerakkan kamera dan menjelajahi ruangan 3D dengan kontrol berikut:
W / A / S / D → Bergerak maju, kiri, mundur, dan kanan
Mouse → Mengatur arah pandang kamera (rotasi)
L → Menyalakan dan mematikan lampu
M → Menyalakan dan mematikan efek refleksi lantai
ESC → Keluar dari program
